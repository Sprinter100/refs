ps, top (top -u %username%) - запущенные процессы
jobs - запущенные программы из терминала
kill <PID> - завершить процесс (флаг -9 - убить процесс)
mkdir -p - создать папку вместе с родителями
"программа" > файл - запись вывода в файл
"программа" >> файл - запись вывода в файл с дозаписью
"программа" 2> файл - запись stderr в файл
"программа" < файл - передать данные программе на вход из файла
"программа" & - запуск в фоновом режиме
"программа1" | "программа2" - stdout программы 1 на stdin программе 2
free -h - кол-во оп. памяти
ls -l -R - подробный вывод всех файлов и папок включая подпапки
which <команда> - какая программа запустится по этой команде
sed - потоковый редактор (прим. cat file.txt | sed 's/word/слово/g' > file2.txt)


Команда find:
-name - с учетом регистра
-iname без учета регистра
-path <путь> - поиск с учётом пути
-mindepth число/ -maxdepth число - мин/макс глубина поиска

Команда grep
-l - Найли файлы c искомым текстом
-L - Файлы, не содержащие искомого текста
-n - выводить номера строк с искомым текстом
-c - вывести кол-во строк с искомым текстом
-A/B/C n - выводит n-ное кол-во строк до/после/вокруг строки с искомым ткестом
-E "regex" - использовать регексы


VIM

:wq (:q!) - Выход  с сохранением (без сохранения)
gg/G - В начало/конец файла
w/e/b - К началу след. слова/к концу слова/ход назад на начало слова
W/E/B - То же, только со СЛОВАМИ (см. word и WORD)
i - режим вставки (esc - выход)
v - режим выделения
yy - скопировать строку
p - вставить
:/текст - найти текст, n - поиск вперед, N - поиск назад
:%s/искать/заменить_на/флаги - замена текста в файле
dd - удалить строку
d<кол-во>d/w - удалить n строк, слов
